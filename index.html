    Project Overview

    项目总览

    课设任务

    本项目基于自设计 MCU 实现 8 点 FFT 运算。输入数据来自 test_ROM/test_vector_in，输出结果写入 verify_RAM/verify_vector_out，最终以 cnt_test 计数结果衡量处理速度。

    设计目标

    1. 正确完成 8 点复数 FFT。
    2. 输入输出均采用 16 位定点数。
    3. 使用 MCU 指令完成全部运算。
    4. 尽可能减少 cnt_test。
    5. 保证综合后 WNS 为正。
    6. 完成上板验证和 ILA 观测。

    技术路线图

    test_ROM 输入MCU 读取 8 点复数数据Q15 定点处理 手工展开 8 点 FFTMUL / ASR / MULQ15 加速写入 verify_RAMILA 观察 cnt_test 与输出

    项目核心策略

    本组不采用通用 N 点 FFT，而是针对 8 点 FFT 任务规模固定的特点，采用手工展开蝶形结构，减少循环、分支和地址计算开销。同时设计面向定点运算的扩展指令，如 MUL、ASR、MULQ15，以降低旋转因子计算周期。
